# Licensing Guide

This project uses a dual-license model for fork-owned changes:

1. **AGPL-3.0-or-later** (community/open-source usage)
2. **Commercial license** (for proprietary/closed-source usage)

## Community Usage (AGPL)

If you use, modify, or deploy this project under the AGPL license, you must
comply with AGPL obligations, including source code availability obligations
for network use/distribution scenarios.

- AGPL text: `LICENSE`

## Commercial Usage

If you need to use this project in a proprietary/closed-source product or want
to avoid AGPL obligations, contact the maintainer for a commercial license.

- Commercial terms template: `COMMERCIAL_LICENSE.md`

## Upstream and Third-Party Code

This repository contains code derived from upstream MIT-licensed work. Upstream
and third-party attribution/licensing remains in effect for those components.

- Upstream attribution: `NOTICE`
- Third-party terms: `THIRD_PARTY_LICENSES.md`

## Trademark

Code licensing does not grant trademark rights.

- Trademark policy: `TRADEMARK.md`
